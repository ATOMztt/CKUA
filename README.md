# CKUA: Leveraging Counterfactual Distillation and Contrastive Adaptation for Robust Public Speaking Anxiety Detection with Missing Modalities

This repo implements the CKUA framework for the following paper: "CKUA: Leveraging Counterfactual Distillation and Contrastive Adaptation for Robust Public Speaking Anxiety Detection with Missing Modalities". 
The project is implemented using the PyTorch framework.
