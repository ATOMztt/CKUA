# CKUA: Leveraging Counterfactual Distillation and Contrastive Adaptation for Robust Public Speaking Anxiety Detection with Missing Modalities

The project is a partial implementation of the paper "CKUA: Leveraging Counterfactual Distillation and Contrastive Adaptation for Robust Public Speaking Anxiety Detection with Missing Modalities". This project is implemented using the PyTorch framework.
